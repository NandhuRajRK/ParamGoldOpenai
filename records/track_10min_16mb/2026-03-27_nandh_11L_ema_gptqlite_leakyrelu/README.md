## 11L EMA + GPTQ-lite + LeakyReLU^2 + QAT@0.15

This folder is a SOTA-chasing scaffold built from the public 2026-03-22 record family and one activation change from the current public leader.

Base:
- `2026-03-22_11L_EMA_GPTQ-lite_warmdown3500_QAT015_1.1233`

Added change:
- replace `relu^2` with `LeakyReLU(0.5)^2`

Portability changes:
- fallback to PyTorch SDPA when `flash_attn_interface` is unavailable
- use `torch.no_grad()` instead of `torch.inference_mode()` in eval paths
- clone rotary cache tensors before reuse
- allow `USE_TORCH_COMPILE=0`

## Intended Stack

- 11 transformer layers, 512 dim, 8 heads, 4 KV heads
- 3x MLP (1536 hidden)
- XSA on last 4 layers
- partial RoPE 16/64
- LN scale
- Shared Value Embedding on late layers
- BigramHash (2048 x 128)
- EMA (0.997)
- tight SWA
- GPTQ-lite int6 export
- late QAT at LR scale < 0.15
- warmdown 3500
- LeakyReLU(0.5)^2

## Why This Branch

The earlier portable 10-layer stack is useful on cheap proxy hardware, but it is too far from the current public frontier to be a serious SOTA attempt.

This branch keeps the stronger 11-layer EMA/GPTQ-lite/QAT recipe and adds the cheapest high-signal delta from the current public leader: `LeakyReLU^2`.
